# 放映廳 (Theater) UI/UX 優化方案

本文件針對目前「放映廳」頁面字體偏小、配色過於沈悶、以及片單展示邏輯不夠美觀等問題，提出具體的視覺與佈局改良建議。

---

## 1. 視覺系統改良 (解決「太小、太黑」)

核心理念是從「死黑 (#000)」轉向具備深度的 **「深炭灰 (Deep Charcoal)」**，透過提昇海拔感 (Elevation) 與字體等級 (Font Scale) 來改善可讀性。

### 1.1 配色方案 (Color Palette)
| 項目 | 目前數值 (估計) | **建議新數值** | 說明 |
| :--- | :--- | :--- | :--- |
| **底層背景 (Base)** | `#0a0b0c` | **`#121418`** | 具備微量藍調的深色，減少 OLED 拖影與視覺疲勞。 |
| **區塊容器 (Section)** | `rgba(20, 20, 20, 0.5)` | **`#1c1e22`** | 讓內容區塊從背景中明顯「浮出」。 |
| **卡片背景 (Card)** | `rgba(8, 8, 10, 0.34)` | **`#24272c`** | 基礎卡片色。Hover 時提升至 `#2d3137`。 |
| **邊框 (Border)** | `rgba(255, 255, 255, 0.08)`| **`rgba(255, 255, 255, 0.12)`** | 提昇邊界定義，但保持柔和。 |
| **主要文字 (Primary)** | `rgba(255, 255, 255, 0.85)` | **`#f0f2f5`** | 增加文字與背景的對比純度。 |
| **次要文字 (Muted)** | `rgba(255, 255, 255, 0.45)` | **`rgba(255, 255, 255, 0.65)`** | 確保說明文字具備足夠可讀性。 |

### 1.2 字體系統 (Typography)
| 項目 | 目前數值 | **建議新數值** | 說明 |
| :--- | :--- | :--- | :--- |
| **最小字體 (Meta)** | `0.68rem` (11px) | **`0.82rem` (13px)** | 嚴禁低於此值，確保標籤與元數據清晰。 |
| **內文 (Body)** | `0.84rem` (13.5px) | **`0.95rem` (15px)** | 標準閱讀體感，增加掃視舒適度。 |
| **標題 (Heading)** | `1.8rem` (28px) | **`2.2rem` (35px)** | 拉大層次，增加「放映廳」的儀式感。 |
| **行高 (Line Height)** | `1.45` ~ `1.6` | **`1.75`** | 增加文字垂直間距，減少壓迫感。 |

## 2. 佈局重構 (解決「展示不喜歡」)

目前的頁面為單一垂直長清單，建議改為 **「模組化導覽」**。

### 2.1 引入 Tab 分頁
將內容拆解為三大部分，避免使用者一進入就被過量資訊淹沒：
- **概覽 (Overview)**：Hero、共享標籤、當前主放映廳推薦。
- **策展 (Curated Lists)**：成員建立的完整片單。
- **板塊 (Board)**：留言板與動態。

### 2.2 片單展示：橫向轉盤 (Carousel)
針對「建議先看 (Recommended)」與「共同片單 (Watchlist)」，使用 **橫向滑動卡片**：
- **優點**：節省垂直空間，符合影視平台 (Netflix/Letterboxd) 直覺。
- **設計**：海報比例建議改為 **2:3 (160x240)**，資訊（標題/共鳴度）可疊加在海報底部邊緣。

### 2.3 成員片單：海報牆 (Mood Board)
將目前的「列表式」改為「視覺主導」：
- 使用 **疊加的海報堆 (Stacked Posters)** 代表一個片單，點擊後展開。
- 註解 (Note) 改為懸停或點擊海報後顯示，減少靜態頁面的文字負擔。

---

## 3. 互動體驗優化

- **編輯器分離 (Decoupled Composer)**：
  - 將「建立片單」、「搜尋電影」等互動表單移至 **Modal (彈窗)** 或 **Slide-over (側邊滑出)**。
  - 主頁面保持「純粹欣賞與閱讀」模式，不再被輸入框打斷視覺流暢度。
- **視覺化共鳴度**：
  - 將「n 人共振」從文字標籤改為 **色彩進度條** 或 **圓環百分比**，直觀呈現該片與該廳的「基因契合度」。
